Click on the Map to freeze the Grid Reference display and view photos for that square.

You can choose whether clicking the map will show photos in that gridsquare or centisquare (which is great with the new centisquare coverage layer)
  • Click on the map once to freeze the Grid Reference display, and load thumbnails
  • Click anywhere on the map to change location new!
  • Click the same location again or the X beside the grid reference to unfreeze the Grid Reference Display.

Map now includes multiple zoom levels!

Do not use the browser back button.
